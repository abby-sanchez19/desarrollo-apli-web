Mi Primer Commit en Git

  1. Crear una carpeta Vacia
  2. En vs code, abrir la carpeta creada (open folder)
  3. Usar el menú de Terminal y seleccionar la opción New Terminal
  4. En la terminal, escribir el comandogit init
  5. Se Crea una Subcarpeta 01-git y dentro un archivo index.html
  6. Verifico el estado actual de git, con el comandogit status
  7. Ejecuto el comando git add .para añadir todos los archivos al Staging Area(encuadre)
  8. Mandar el commit (tomar la foto) para guardar los cambios en el repositorio usando el comando git commit -m "Mensaje descriptivo de mi commit"
